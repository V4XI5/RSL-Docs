
GetObject
********************************************************


**Arguments:**

- ``string`` **Name**: The name of the object to find.

Alternative:

- ``unsigned int`` **Handle**: The handle of the object to find.

**Returns:**

- `Object`_ or nil depending on if an object of the given name or handle is found.
